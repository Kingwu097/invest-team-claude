# Design System — Invest Team Claude

## Product Context
- **What this is:** Multi-Agent 投资理财专家团队 Dashboard，展示 AI 分析师的辩论过程和共识报告
- **Who it's for:** 开发者本人日常投资辅助 + 技术博客/演讲展示
- **Space/industry:** FinTech / AI Agent 可视化
- **Project type:** Web App (Dashboard)

## Aesthetic Direction
- **Direction:** Industrial Precision（工业精密）
- **Decoration level:** Minimal — 排版和数据本身就是装饰，零装饰性元素
- **Mood:** 专业交易终端遇到现代 AI 产品。沉稳、数据密集但清晰，用精确的排版和克制的色彩传递专业感。不是 Bloomberg 的上世纪风格，也不是消费级 App 的圆润可爱。像看一场投资辩论直播，有节奏感和叙事性。

## Typography
- **Display/Hero:** Satoshi — 几何无衬线，现代但不冰冷，用于标题和股票名称
  - Loading: `https://api.fontshare.com/v2/css?f[]=satoshi@400,500,700&display=swap`
- **Body:** DM Sans — 可读性极强，中等 x-height，用于正文和 Agent 分析内容
  - Loading: Google Fonts `DM+Sans:wght@400;500;600`
- **UI/Labels:** Same as body (DM Sans 500)
- **Data/Tables:** Geist Mono — 等宽字体，tabular-nums，用于数字、指标、信心度、Token 消耗
  - Loading: Google Fonts `Geist+Mono:wght@400;500`
- **Code:** Geist Mono
- **Chinese fallback:** -apple-system, "PingFang SC", "Microsoft YaHei", sans-serif
- **Scale:**
  - 3xl: 36px / Satoshi 700 — 股票名称、主标题
  - 2xl: 24px / Satoshi 700 — 共识评级、section 标题
  - xl: 20px / Satoshi 700 — Agent 评级
  - lg: 18px / Satoshi 600 — 副标题
  - md: 15px / DM Sans 400 — 正文
  - sm: 13px / DM Sans 500 — UI 标签、按钮、时间线内容
  - xs: 12px / DM Sans 400 — 次要信息、时间线细节
  - 2xs: 11px / Geist Mono 400 — 时间戳、section 标题（uppercase + letter-spacing 0.08em）
  - data-lg: 28px / Geist Mono 500 — 大数字（涨跌幅）
  - data-md: 15px / Geist Mono 400 — 指标行
  - data-sm: 11px / Geist Mono 400 — 时间戳

## Color
- **Approach:** Restrained — 1 accent + 语义色，颜色稀少且有意义
- **Dark mode (default):**
  - `--bg`: #0C0E14 — 页面背景（近乎纯黑的深蓝灰）
  - `--surface`: #161922 — 卡片/面板背景
  - `--surface-hover`: #1C2030 — 悬停态
  - `--border`: #252A36 — 微妙分隔线
  - `--text-primary`: #E8ECF4 — 主要文本（高对比白）
  - `--text-secondary`: #7C859B — 次要文本
  - `--text-muted`: #4A5068 — 弱化信息（时间戳、标签）
  - `--accent`: #3B82F6 — 强调色（按钮/链接/选中态，克制的蓝色）
  - `--accent-hover`: #2563EB
  - `--bull`: #22C55E — 看多/上涨（绿色，中国市场习惯）
  - `--bear`: #EF4444 — 看空/下跌（红色）
  - `--neutral-color`: #F59E0B — 中性评级（琥珀色）
  - `--warning`: #F59E0B — 高分歧警告（琥珀色复用）
- **Light mode:**
  - `--bg`: #F8F9FC
  - `--surface`: #FFFFFF
  - `--surface-hover`: #F1F3F8
  - `--border`: #E2E5ED
  - `--text-primary`: #111827
  - `--text-secondary`: #6B7280
  - `--text-muted`: #9CA3AF
  - Accent/Bull/Bear/Neutral: same hex values
- **Semantic:**
  - Success: #22C55E (same as bull)
  - Warning: #F59E0B (same as neutral/warning)
  - Error: #EF4444 (same as bear)
  - Info: #3B82F6 (same as accent)
- **Design rationale:** 深色背景让红绿信号色跳出来（看多/看空一目了然），克制的蓝色强调色不跟交易信号色冲突。

## Spacing
- **Base unit:** 4px
- **Density:** Comfortable — 投资数据需要呼吸空间
- **Scale:** 2xs(2) xs(4) sm(8) md(16) lg(24) xl(32) 2xl(48) 3xl(64)

## Layout
- **Approach:** Hybrid — 主工作区严格网格，圆桌概览自由布局
- **Grid:** Sidebar(220px) + Main(1fr), Main 内部单列流式
- **Max content width:** 1440px
- **Border radius:**
  - sm: 4px — 按钮、输入框、sidebar item
  - md: 8px — 卡片、面板
  - lg: 12px — Dashboard 外框
  - full: 9999px — 徽章 (badge)
- **Sidebar:** 固定 220px 左侧边栏，深色 surface 背景，含历史记录列表和快捷分析输入
- **Main content:** 垂直流式布局：圆桌概览 → 时间线 → 信心度图

## Motion
- **Approach:** Intentional — 有目的的动画，传达"AI 正在实时思考"
- **Easing:** enter(ease-out) exit(ease-in) move(ease-in-out)
- **Duration:** micro(50-100ms) short(150-250ms) medium(250-400ms) long(400-700ms)
- **Key animations:**
  - 时间线节点入场: slide-in(8px) + fade, 0.4s ease-out, stagger 100ms — SSE 事件逐个出现的节奏感
  - 共识达成 dot: glow shadow pulse (box-shadow 0 0 8px rgba(34,197,94,0.4))
  - 等待中 dot: opacity pulse 1.5s ease-in-out infinite
  - 信心度折线图: 平滑 CSS transition 0.3s
  - 圆桌概览连线: SVG stroke-dashoffset 描边动画

## Component Patterns
- **Badge variants:** bull(绿底绿字) / bear(红底红字) / neutral(琥珀底琥珀字) / divergence(琥珀边框+⚡)
- **Button variants:** primary(accent蓝) / secondary(surface+border) / ghost(transparent) / bull(绿) / bear(红)
- **Alert variants:** success(绿) / warning(琥珀) / error(红) / info(蓝) — 均为 8% opacity 背景 + 20% opacity 边框
- **Timeline dot colors:** data(accent蓝) / bull(绿) / bear(红) / neutral(琥珀) / debate(琥珀) / consensus(绿+glow)
- **Challenge block:** bg(--bg) + left-border 2px warning色 + sm radius 右侧

## AI Slop Avoidance
Never use in this project:
- Purple/violet gradient backgrounds
- 3-column icon-in-circle feature grids
- Centered everything layout
- Uniform bubbly border-radius
- Decorative blobs or wavy SVG dividers
- Generic hero sections
- Emoji as design elements in UI (OK in reports generated by AI)
